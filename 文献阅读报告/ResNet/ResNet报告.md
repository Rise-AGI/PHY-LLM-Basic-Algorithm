# 论文详细报告

## 用于图像识别的深度残差学习
**Deep Residual Learning for Image Recognition**

- **作者：** Kaiming He, Xiangyu Zhang, Shaoqing Ren, Jian Sun
- **机构：** Microsoft Research
- **发表：** CVPR 2016
- **荣誉：** ILSVRC 2015 分类、检测、定位冠军；COCO 2015 检测、分割冠军

---

## 一、研究背景与核心问题

### 1.1 深度的重要性

深度卷积神经网络在图像分类任务上取得了一系列突破性进展。研究表明，网络深度对性能至关重要——从 VGG 的 16 层到 GoogleNet 的 30 层，更深的网络带来了更好的特征表示。

### 1.2 退化问题（Degradation Problem）

然而，简单地堆叠更多层会引发一个反直觉的现象：

> **网络越深，训练误差反而越高——这不是过拟合，而是优化困难。**

如图1所示，56层的"plain网络"在 CIFAR-10 上的训练误差**高于** 20 层网络，验证误差同样更差。

**这一现象令人困惑：** 理论上，深层网络是浅层网络的超集（多出的层只需学习恒等映射即可），不应该更差。但实验证明，现有优化器无法找到这样的解。

### 1.3 研究问题

> **如何让更深的网络至少不比浅层网络差，甚至更好？**

---

## 二、核心方法：残差学习

### 2.1 基本思想

**传统做法：** 让堆叠的层直接拟合目标映射 $\mathcal{H}(\mathbf{x})$

**残差学习：** 让这些层改为拟合**残差映射** $\mathcal{F}(\mathbf{x}) := \mathcal{H}(\mathbf{x}) - \mathbf{x}$

则原始映射变为：
$$\mathcal{H}(\mathbf{x}) = \mathcal{F}(\mathbf{x}) + \mathbf{x}$$

**为什么这样更容易优化？**

- 如果恒等映射是最优解，让残差趋向零比用非线性层拟合恒等映射**容易得多**
- 残差函数的参考点是输入 $\mathbf{x}$，优化方向更明确
- 实验证实：残差函数的响应普遍较小，说明恒等映射是合理的预条件

### 2.2 残差块（Building Block）

```
输入 x
  │
  ├──────────────────┐（shortcut连接）
  │                  │
weight layer (conv)  │ 恒等映射
     relu            │
weight layer (conv)  │
  │                  │
  └──────── + ───────┘
            │
           relu
         输出 y = F(x) + x
```

**数学表达：**

$$\mathbf{y} = \mathcal{F}(\mathbf{x}, \{W_i\}) + \mathbf{x} \quad \text{（维度相同时）}$$

$$\mathbf{y} = \mathcal{F}(\mathbf{x}, \{W_i\}) + W_s\mathbf{x} \quad \text{（维度不同时，用} 1\times1 \text{卷积对齐）}$$

**关键特性：**
- shortcut 连接**不增加任何参数**，不增加计算复杂度
- 整个网络仍可端到端用 SGD + 反向传播训练
- 无需修改任何现有框架（Caffe 等）

## 三、网络架构设计

### 3.1 设计原则（参考 VGG）

1. 相同输出特征图尺寸的层，使用相同数量的滤波器
2. 特征图尺寸减半时，滤波器数量加倍（保持每层时间复杂度一致）
3. 用步长为 2 的卷积代替池化做下采样
4. 末端用全局平均池化 + 全连接层（1000类 softmax）

### 3.2 Plain 网络 vs ResNet

| 组件 | Plain 网络 | ResNet |
|------|-----------|--------|
| 卷积层 | 相同 | 相同 |
| shortcut | 无 | 每两个 $3\times3$ 层加一个 |
| 参数量 | 相同 | 相同（恒等 shortcut 无参数）|
| 可优化性 | 退化问题严重 | 深层仍可优化 |

### 3.3 各版本 ResNet 架构（表1）

| 网络 | 层数 | 构建块类型 | FLOPs |
|------|------|----------|-------|
| ResNet-18 | 18 | 基本块（2层）×8 | $1.8\times10^9$ |
| ResNet-34 | 34 | 基本块（2层）×16 | $3.6\times10^9$ |
| ResNet-50 | 50 | 瓶颈块（3层）×16 | $3.8\times10^9$ |
| ResNet-101 | 101 | 瓶颈块（3层）×33 | $7.6\times10^9$ |
| ResNet-152 | 152 | 瓶颈块（3层）×50 | $11.3\times10^9$ |

**152层 ResNet 的复杂度仍低于 VGG-16/19（15.3/19.6 GFLOPs）！**

### 3.4 瓶颈块（Bottleneck Block）

针对较深的网络（50层及以上），用瓶颈设计降低训练成本：

```
输入（256维）
     │
  1×1 conv, 64     ← 降维
     relu
  3×3 conv, 64     ← 核心计算（在低维进行）
     relu
  1×1 conv, 256    ← 升维还原
     │
   + x（shortcut）
     relu
输出（256维）
```

**优势：** 两种设计时间复杂度相近，但瓶颈块参数效率更高，适合极深网络。

## 四、训练设置

### 4.1 数据预处理与增强

- 图像短边随机采样自 $[256, 480]$ 做尺度增强
- 随机裁剪 $224\times224$，随机水平翻转
- 减去像素均值，使用标准颜色增强
- 每次卷积后、激活前使用**批归一化（BN）**

### 4.2 优化超参数

| 超参数 | 值 |
|--------|---|
| 优化器 | SGD |
| Mini-batch | 256 |
| 初始学习率 | 0.1（误差平台时除以10）|
| 权重衰减 | 0.0001 |
| 动量 | 0.9 |
| 最大迭代数 | $60\times10^4$ |
| Dropout | 不使用 |

### 4.3 测试策略

- **对比实验：** 标准 10-crop 测试
- **最优结果：** 全卷积形式，多尺度（224/256/384/480/640）平均

## 五、主要实验结果

### 5.1 ImageNet 分类

#### 退化问题验证（Plain 网络）

| 网络 | Top-1 错误率 |
|------|------------|
| Plain-18 | 27.94% |
| Plain-34 | **28.54%**（更深反而更差！）|

#### 残差网络完全解决退化问题

| 网络 | Top-1 错误率 | 对比 |
|------|------------|------|
| ResNet-18 | 27.88% | ≈ Plain-18 |
| ResNet-34 | **25.03%** | 比 Plain-34 低 **3.5%** |

**结论：** 34 层 ResNet 优于 18 层 ResNet，成功从深度中获益。

#### 与最优方法对比（单模型，表4）

| 方法 | Top-1 错误率 | Top-5 错误率 |
|------|------------|------------|
| VGG (v5) | 24.4% | 7.1% |
| PReLU-net | 21.59% | 5.71% |
| BN-Inception | 21.99% | 5.81% |
| ResNet-50 | 20.74% | 5.25% |
| ResNet-101 | 19.87% | 4.60% |
| **ResNet-152** | **19.38%** | **4.49%** |

#### 集成模型（表5）

| 方法 | Top-5 测试错误率 |
|------|----------------|
| VGG (ILSVRC'14) | 7.32% |
| GoogLeNet (ILSVRC'14) | 6.66% |
| PReLU-net | 4.94% |
| **ResNet（ILSVRC'15）** | **3.57%** |

> ResNet 集成以 **3.57%** 的 Top-5 错误率赢得 ILSVRC 2015 冠军。

### 5.2 Shortcut 类型消融（表3）

| Shortcut 方式 | Top-1 错误率 | Top-5 错误率 |
|-------------|------------|------------|
| (A) 零填充（无参数）| 25.03% | 7.76% |
| (B) 升维用投影，其余恒等 | 24.52% | 7.46% |
| (C) 全部用投影 | 24.19% | 7.40% |

**结论：** A/B/C 差距很小，说明**投影 shortcut 对解决退化问题不是必须的**；恒等 shortcut 足够，且更经济（不增加参数）。

### 5.3 CIFAR-10 实验

在 CIFAR-10 上验证了与 ImageNet 相同的规律：

| 网络 | 层数 | 参数量 | 测试错误率 |
|------|------|--------|----------|
| Highway | 19 | 2.3M | 7.54% |
| ResNet | 20 | 0.27M | 8.75% |
| ResNet | 32 | 0.46M | 7.51% |
| ResNet | 44 | 0.66M | 7.17% |
| ResNet | 56 | 0.85M | 6.97% |
| **ResNet** | **110** | **1.7M** | **6.43%** |
| ResNet | 1202 | 19.4M | 7.93%（过拟合）|

**1202层网络训练误差 <0.1%，证明优化无障碍，但在小数据集上过拟合。**

### 5.4 目标检测（Faster R-CNN backbone）

用 ResNet-101 替换 VGG-16 作为 Faster R-CNN 骨干网络：

**PASCAL VOC：**

| 骨干网络 | mAP（VOC 07 test）|
|---------|-----------------|
| VGG-16 | 73.2% |
| **ResNet-101** | **76.4%** |

**MS COCO：**

| 骨干网络 | mAP@0.5 | mAP@[0.5,0.95] |
|---------|---------|----------------|
| VGG-16 | 41.5% | 21.2% |
| **ResNet-101** | **48.4%** | **27.2%（+28% 相对提升）**|

## 六、深层分析

### 6.1 层响应分析（图7）

对 CIFAR-10 各层 $3\times3$ 卷积输出（BN后、ReLU前）的标准差分析：

- **ResNet 的层响应普遍小于对应 Plain 网络**
- 更深的 ResNet（110层）响应更小
- 这证实了残差函数确实比原始映射更接近零，恒等映射是合理的预条件

**深层含义：** 网络大部分层只需对特征做**微小调整**，而非完整重建，这使优化更简单。

### 6.2 为什么 Plain 网络退化？

作者排除了梯度消失的可能性（BN 保证了前向/反向信号的健康范数），推测：

> **深层 Plain 网络可能有指数级慢的收敛速度，导致训练误差难以降低。**

这一问题的根本原因留待后续研究。

### 6.3 极深网络探索（1202层）

- 训练误差 <0.1%（优化完全没有问题）
- 测试误差 7.93%（比 110 层差，因为在小数据集 CIFAR-10 上过拟合）
- 说明残差学习的**优化能力**已不是瓶颈，数据量成为制约因素

## 七、与相关工作的对比

### 7.1 vs Highway Networks

| 维度 | Highway Networks | ResNet |
|------|----------------|--------|
| Shortcut 类型 | 门控（有参数，数据依赖）| 恒等（无参数）|
| Shortcut 是否可关闭 | 可以（门→0）| 永不关闭 |
| 极深时精度提升 | 未展示（>100层）| 显著提升 |
| 参数开销 | 更大 | 更小 |

### 7.2 vs VGG

- ResNet-152（11.3 GFLOPs）复杂度低于 VGG-16（15.3 GFLOPs）
- 精度大幅领先（4.49% vs 7.1% Top-5）

## 八、局限性与后续影响

### 8.1 本文局限

- 1202 层网络在小数据集上过拟合，强正则化（Dropout/Maxout）的结合有待研究
- 退化问题的理论根本原因未完全解释

### 8.2 后续工作

本文直接催生了大量重要工作：

| 后续工作 | 关系 |
|----------|------|
| **DenseNet**（Huang et al., 2017）| 将 shortcut 推广为密集连接 |
| **ResNeXt**（Xie et al., 2017）| 聚合变换 + 残差 |
| **Wide ResNet**（Zagoruyko, 2016）| 用宽度换深度 |
| **Pre-activation ResNet**（He et al., 2016）| BN/ReLU 放在卷积前 |
| **SENet**（Hu et al., 2018）| 在残差块中加注意力 |
| **Transformer 中的残差连接** | Attention Is All You Need 直接引用本文 shortcut 思想 |

## 九、结论

### 主要贡献

1. **发现并系统研究了退化问题**：深层网络在训练时比浅层网络更差，非过拟合所致
2. **提出残差学习框架**：通过 shortcut 连接让网络拟合残差而非原始映射
3. **证明深度的价值**：152 层 ResNet 在 ImageNet 上达到 3.57% Top-5 错误率
4. **无额外开销**：恒等 shortcut 零参数、零计算增量，实现简单
5. **强泛化性**：在分类、检测、分割、定位多个任务上均获 ILSVRC & COCO 2015 冠军

### 核心公式总结

$$\boxed{\mathbf{y} = \mathcal{F}(\mathbf{x}, \{W_i\}) + \mathbf{x}}$$

> **一句话总结：** 与其让深层网络从零开始学习每一层的映射，不如让它只学习相对于输入的"残差"——这一简单的重新表述，使得训练数百层的深度网络成为可能，彻底改变了深度学习的工程实践，ResNet 至今仍是计算机视觉领域最广泛使用的骨干网络之一。
